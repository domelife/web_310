11. 박스속성

박스속성은 웹페이지 레이아웃 구성 시 가장 중요한 요소들이다.

Margin, Padding, Width/Height의 속성
  • margin - 여백
  • padding - 영역
  • width, height - 크기

  1. 작성법 :

    • 전체적용 - 50px 하나의 숫자로 4군데 모두 적용
    • 각자적용 - 윗쪽 오른쪽 아래쪽 왼쪽 순서임(시계방향) / 모두 단위와 함께 작성 중간에 쉼표 쓰지 않음
      위아래 좌우 / 두개의 숫자로 첫 번째 숫자 - 위와 아래 같이 적용, 두 번째 숫자 - 좌와 같이 적용
    • 부분적용
      • margin-left : 왼쪽 여백
      • margin-right: 오른쪽 여백
      • margin-top : 위쪽 여백
      • margin-bottom: 아래쪽 여백
      • padding-left: 왼쪽 영역
      • padding-right: 오른쪽 영역
      • padding-top: 위쪽 영역
      • padding-bottom: 아래쪽 영역

  2. 길이 측정 :

    • width : 가로크기값
      width + (padding + border) * 2

    • height : 세로크기값
      height + (padding + border) * 2

Border의 속성

 박스의 테두리를 말하며, 가로/세로 값에 영향을 준다.
색상, 굵기, 모양을 한번에 모두 표기 시 보인다.
  1. 작성법
    • border : black 2px solid (순서와 관계 없음)
    • 4면 모두 한번에 나오거나 지정해도 상관없다. (border-left,right,top,bottom)
  2. 선의 종류
    • dashed(긴 점선), dotted(짧은 점선), double(두줄 선), 액자선(groove, ridge, inset, outset), none(없음)

Background 속성

배경색, 배경이미지, 배경반복, 배경위치, 배경고정 등을 설정한다.

  • background-color: 색깔이름, RGB, RGBA로 표기 가능 / transparant(투명색)
  • background-image: url("인터넷 위치 주소 기재")
  • background-repeat:
    • repeat(가로, 세로 모두 반복)
    • repeat-x(가로반복), repeat-y(세로반복)
    • no-repeat(반복없음)
  • background-position:
    • 가로위치, 세로위치, 생략 시 기본값은 (left, top)이다.
    • 숫자로 지정 시 왼쪽 상단 기준에서 시작됨. 가로숫자, 세로숫자
    • 위치명으로도 기재 가능(0,0)
  • background-attatchment: 배경이미지를 고정시킬지 스크롤할지 결정(scroll, fixed)
  • background-size
    • 100% - 이미지를 넣었을 때, 화면을 비율 상관없이 억지로라도 꽉 채움
    • contain - 배경이미지를 잘리는 부분 관계없이 이미지 우선으로 보임(빈 공간이 있든 말든 이미지 보존)
    • cover - 배경이미지와 관계없이 화면을 꽉 채움
