선택자의 종류

html 파일을 css를 통해 꾸밀 수 있고, 꾸미기 위해서는 지정된 선택을 해야한다.

작성법

선택자 { 속성명 : 속성값; }

선택자 종류 작성 방법 설명
전체 선택자 * HTML 내부의 모든 태그를 불러서 설정한다.(건 by 건)
태그 선택자 태그명 HTML 내부의 '불려진' 모든 태그를 설정한다.
아이디 선택자 #아이디이름 속성값 ID의 이름을 불러서 특정 ID만 설정한다.
클래스 선택자 .클래스명 속성값 class의 이름을 불러서 특정 클래스만 설정한다.
id와 class의 차이점:
id(이름)는 한페이지에 한번만 사용할 수 있다.(중복사용이 안됨 - 한번 사용한 아이디는 두번 사용하는거 안됨)
그러나 다른 아이디의 이름을 적용하면 또 사용 가능(웹페이지의 헤더, 푸터와 같은 개념. class로 지정하는것보다 더 포괄적인 범위임) class는 중복적용이 가능하다. 또한 한 클래스에 여러 클래스명 사용 가능하다. 클래스명(별명, 한글은 안됨)을 한칸 띄어쓰기로 여러 클래스명을 부여할 수 있다.
중복선택자 , 여러개의 선택자를 나열해서 사용할 수 있다.
중복선택자 붙여쓰기(태그 이름을 쓰고 붙여서 .클래스명) 지정된 태그 중 해당 클래스이름만 부름
후손선택자 선택자 선택자 부모자 내부에 있는 요소를 모두 부른다.
자손선택자 부모자>자손선택자 다이렉트 자손자에 한해서 설정 가능하다.
