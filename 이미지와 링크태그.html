이미지태그

웹문서에 이미지를 불러오는 요소태그
인라인태그로 줄바꿈이 되지 않음
닫힘태그(짝궁)요소 없음
이미지 태그와 함께 사용하는 필수속성 태그가 존재함.

이미지 태그 속성들

  • src = "이미지의 경로지정"
  • width = "가로사이즈 숫자지정(px단위 자동설정)"
  • height = "세로사이즈 숫자지정(px단위 자동설정)"
  • alt = "대체텍스트 속성"

alt 속성은 웹 접근성(다양한 인류와 연령대 등등을 다 포함해서 모든 사람이 자유롭게 사용할 수 있게함)을 사용할 때 필수작성요소

이미지 태그 예제

 뷸특정한 사진을 공간을 채우기 위해 사용하는 사이트
https://picsum.photos/200/300 - 가로 200 * 세로 300 사이즈의 사진 등록



링크태그

웹문서에서 다른 페이지나 웹페이지 내부의 특정위치로 이동할 때 사용하는 anchor 태그 - a 태그
닫힘태그와 함께 사용하고 줄바꿈이 없는 인라인속성이다.

앵커태그속성

  • href = "위치경로 주소 작성"
  • target = "_blank" (새창열기)

링크태그 예제
  • 네이버로 바로가기
  • 네이버 새창 열기
